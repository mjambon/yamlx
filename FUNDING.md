# Funding and Licensing

## Current license: AGPL

YAMLx is currently released under the
[GNU Affero General Public License v3 (AGPL)](LICENSE). The AGPL
requires that any software that incorporates YAMLx — including software
offered as a network service — must itself be released under the AGPL.
For many commercial or proprietary projects this is not acceptable.

## Fundraiser: unlock ISC for everyone

There is an ongoing fundraiser with the following goal: once the funding
threshold of **$5,000** is reached, the default license for YAMLx will
switch to the [ISC license](https://opensource.org/licenses/ISC), a
permissive two-clause license with no copyleft requirements. At that point
everyone — past, present, and future — will be able to use YAMLx freely
in any project.

Funds will be collected through **GitHub Sponsors**. Progress toward
the goal will be tracked there.

## Donation tiers

| Contribution | Benefit |
|---|---|
| $10 or more | Recognition by name in this file |
| $500 or more | Immediate commercial license + name recognition |

**Commercial license** holders may use YAMLx in proprietary or commercial
software without having to comply with the terms of the AGPL.

To obtain a commercial license, sponsor the project through GitHub Sponsors
at the qualifying tier and then open an issue or send a message to confirm
your contribution. A license will be issued promptly.

## A note for organizations

If your company or team relies on YAMLx in a product or service, please
consider contributing in proportion to the value you get from the software.
Even a modest sponsorship helps sustain development and moves the project
closer to a fully permissive license that benefits everyone.

## Why this model?

Open-source maintainers spend significant time building and maintaining
software that businesses rely on. This funding model allows YAMLx to
remain freely available to the open-source community today while giving
the project a path to a fully permissive license — and giving
commercial users a compliant way to use it in the meantime.
