Module YAMLx.Node

Operations on a single lossless AST node.

Node provides the single-node interface analogous to Value: a type t alias and node-level utilities. For parsing and serialising whole documents use Nodes.

type t = node =
  1. | Scalar_node of {
    1. anchor : string option;
    2. tag : string option;
    3. value : string;
    4. style : scalar_style;
    5. loc : loc;
    6. height : int;
    7. head_comments : string list;
    8. line_comment : string option;
    9. foot_comments : string list;
    }
  2. | Sequence_node of {
    1. anchor : string option;
    2. tag : string option;
    3. items : t list;
    4. flow : bool;
    5. loc : loc;
    6. height : int;
    7. head_comments : string list;
    8. line_comment : string option;
    9. foot_comments : string list;
    }
  3. | Mapping_node of {
    1. anchor : string option;
    2. tag : string option;
    3. pairs : (t * t) list;
    4. flow : bool;
    5. loc : loc;
    6. height : int;
    7. head_comments : string list;
    8. line_comment : string option;
    9. foot_comments : string list;
    }
  4. | Alias_node of {
    1. name : string;
    2. resolved : t Stdlib.Lazy.t;
    3. loc : loc;
    4. height : int;
    5. head_comments : string list;
    6. line_comment : string option;
    7. foot_comments : string list;
    }
    (*

    Alias for YAMLx.node. Exposes the constructors under the Node namespace so that Node.Scalar_node, Node.Mapping_node, etc. work alongside YAMLx.Scalar_node.

    *)
val has_comments : t -> bool

has_comments node is true if node or any of its descendants carries at least one comment (head, line, or foot).

val pp : Stdlib.Format.formatter -> t -> unit

Pretty-printer for use with Format and %a.

val show : t -> string

Return a human-readable representation of a node as a string.
