# yaml-test-suite

This directory contains a vendored snapshot of the test cases from
[yaml/yaml-test-suite](https://github.com/yaml/yaml-test-suite),
a comprehensive test suite for YAML parsers maintained by the YAML community.

The `src/` subdirectory holds the individual test case directories, each
containing input YAML and expected output files.
